# 小红书内容创作最佳实践

本文档提供小红书平台内容创作的最佳实践指南，帮助提升内容质量和用户互动。

**⚠️ 重要提示**：
- 本文档内容仅供参考，**不应主动应用到用户内容中**
- **话题标签**：仅在用户明确要求添加标签时才添加，不要主动建议或添加
- **内容优化**：不要擅自修改用户提供的内容，除非用户明确要求优化
- **使用场景**：当用户询问如何写出更好的小红书内容时，可参考此文档提供建议

## 内容创作原则

### 1. 真实性
- 分享真实体验和感受
- 避免过度夸张或虚假宣传
- 使用真实的产品体验照片或描述

### 2. 实用性
- 提供有价值的信息或建议
- 包含具体的操作步骤或技巧
- 解决用户实际问题

### 3. 可读性
- 使用清晰的结构和分段
- 避免过长的段落
- 善用emoji和符号（适度）

### 4. 互动性
- 鼓励用户评论和分享
- 提出问题引发讨论
- 及时回复用户评论

## 普通笔记最佳实践

### 内容结构建议
```
【钩子】吸引人的开头（1-2行）
【主体】核心内容（5-7行）
【收尾】总结或互动引导（1-2行）
```

### 开头技巧
- 使用惊叹句或疑问句
- 制造悬念或好奇心
- 直击痛点或需求

**示例**：
- ❌ "今天分享一个护肤心得"
- ✅ "用了这个方法，黑头真的少了！"

### 表达技巧
- 使用短句，每句话不超过20个字
- 适当使用emoji增加视觉吸引力（但不要过度）
- 关键信息用【】或「」标注

**示例**：
```
发现了一家【人均50元】的宝藏餐厅！
环境超好✨服务贴心👍
强烈推荐他家的招牌菜！
下次还会再来💕
```

### 话题标签使用
- 添加2-5个相关话题标签
- 包含热门话题提高曝光
- 使用地点标签增加本地流量

**标签格式**：`#话题名称`

**示例**：
```
#上海美食 #探店 #人均50元 #周末去哪儿
```

## 长文笔记最佳实践

### 内容结构建议
```
【标题】吸引人且包含关键词
【开头】简短引入（50字内）
【目录】如果内容较长，列出要点
【正文】详细内容，分段清晰
【图片文字】详细信息，转化为图片展示
【正文摘要】亮点总结+互动引导
```

### 标题撰写技巧
- 包含核心关键词
- 控制在15-25字
- 使用数字、符号增加吸引力

**标题模板**：
- 「数字型」：《7天XXX，效果显著！》
- 「方法型」：《如何XXX？这个方法太好用了》
- 「对比型」：《XXX vs XXX，哪个更值得？》
- 「避坑型」：《XXX避坑指南，建议收藏！》

### 图片文字内容设计

图片文字应该是详细的、可独立阅读的内容：

**结构化内容**：
- 使用小标题分段
- 每段3-5行为宜
- 重要信息加粗或高亮

**示例结构**：
```
【第一部分：背景介绍】
简述主题背景和重要性...

【第二部分：详细说明】
具体步骤1：...
具体步骤2：...
具体步骤3：...

【第三部分：注意事项】
⚠️ 注意点1
⚠️ 注意点2

【第四部分：总结】
核心要点总结...
```

### 正文摘要撰写技巧

正文摘要与图片内容的区别：
- **图片内容**：详细教程、完整攻略、深度分析
- **正文摘要**：核心亮点、快速预览、互动引导

**摘要内容要素**：
1. 核心卖点（1-2句话）
2. 适用人群或场景
3. 话题标签（3-5个）
4. 互动引导（提问、征集意见等）

**示例**：
```
✨三天两夜杭州游记来啦！
景点+美食+住宿全攻略，人均1500搞定！

适合：周末短途游、情侣出行、闺蜜旅行

💡 建议收藏，下次出行不迷路！
你们还想看哪个城市的攻略？评论区告诉我～

#杭州旅游 #周末游 #旅行攻略 #闺蜜游 #情侣游
```

## 排版和视觉设计

### 卡片风格选择（普通笔记）
- **简约风格**：适合文字为主的分享
- **清新风格**：适合生活、美食类内容
- **商务风格**：适合职场、干货类内容
- **时尚风格**：适合穿搭、美妆类内容

### 长文排版选择
- **黑底白字**：正式、专业感强
- **白底黑字**：清爽、易读
- **彩色背景**：活泼、年轻化
- **渐变背景**：时尚、吸引眼球

**选择原则**：
- 内容为王，排版服务于内容
- 确保文字清晰可读
- 风格与内容调性匹配

## 内容长度指南

### 普通笔记
- **最少**：3行（过短缺乏信息）
- **适中**：6-8行（最佳长度）
- **最多**：10行（技能硬性限制）

### 长文笔记
- **最少**：100字（技能硬性限制）
- **推荐**：300-800字（适合深度阅读）
- **最多**：1500字（避免过长导致阅读疲劳）

## 内容合规要点

### 禁止内容
- ❌ 虚假宣传、夸大效果
- ❌ 未经授权的品牌推广
- ❌ 涉及政治、色情、暴力内容
- ❌ 侵犯他人隐私或版权
- ❌ 引导站外交易

### 注意事项
- ⚠️ 医疗健康类内容需谨慎，避免诊断性语言
- ⚠️ 金融投资类内容需添加风险提示
- ⚠️ 未成年人相关内容需特别注意保护隐私
- ⚠️ 商业推广需遵守平台规则，标注"广告"

## AutoGLM内容处理原则

**核心原则**：尊重用户原始内容，不要擅自修改或优化。

在使用AutoGLM自动发帖时，对于用户提供的原始内容：

### 基本处理原则
1. **原样使用**：用户提供什么内容就使用什么内容，不要擅自修改
2. **禁止优化**：不要主动优化、美化、重写用户的内容
3. **禁止添加标签**：除非用户明确要求，否则不要添加话题标签
4. **禁止添加emoji**：不要主动添加用户未提供的emoji或符号

### 允许的处理情况
仅在以下情况下可以对内容进行必要调整：
- **用户明确要求优化**：用户说"帮我优化一下"、"帮我写得更吸引人"等
- **技术性调整**：格式化、换行等不改变内容含义的调整
- **长度超限处理**：内容超出限制时，询问用户如何精简

### 必须询问用户的情况
- 内容严重超出字数限制（普通笔记>10行，长文>2000字）
- 内容可能涉及合规问题
- 内容过于简短无法发布（长文<100字）
- 用户未提供必要信息（如标题、正文等）

### 示例：原样使用用户内容（默认行为）

**用户请求**："帮我发一条小红书笔记：今天去了一家新开的火锅店，味道很好，环境也不错，服务员态度很好，价格也合理，推荐大家去试试。"

**正确处理**：
- 识别为普通笔记（内容简短）
- **原样使用**用户提供的内容，不做任何修改
- 按照发布流程直接发布

**错误处理示例（不要这样做）**：
- ❌ 添加emoji："发现宝藏火锅店🔥"
- ❌ 重写内容："味道绝了！环境超赞！"
- ❌ 添加标签："#火锅 #探店 #美食推荐"
- ❌ 任何未经用户要求的优化

### 示例：用户明确要求优化时才可以优化

**用户请求**："帮我发一条小红书笔记，把这段话优化得更吸引人：今天去了一家新开的火锅店，味道很好，环境也不错，服务员态度很好，价格也合理，推荐大家去试试。"

**正确处理**：
- 用户明确要求"优化得更吸引人"，可以进行优化
- 可以参考本文档的最佳实践进行优化
- 优化示例：
  ```
  发现宝藏火锅店🔥
  味道绝了！环境超赞！
  服务小姐姐超温柔💕
  人均才80块！性价比爆棚！
  强烈推荐！周末约起来～
  ```
- **注意**：即使用户要求优化，也不要主动添加话题标签，除非用户明确要求添加
