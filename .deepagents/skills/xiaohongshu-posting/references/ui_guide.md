# 小红书APP UI元素识别指南

本文档提供小红书APP发帖流程中关键UI元素的详细识别指南，帮助AutoGLM准确定位和操作界面元素。

## 主界面

### 红色加号按钮
- **位置**：屏幕底部导航栏，正中央位置
- **外观**：红色圆形或方形按钮，带有白色"+"号图标
- **识别特征**：颜色鲜明，是底部导航栏中唯一的红色元素
- **替代识别方法**：通常是底部5个导航按钮中的第3个（中间位置）

## 发布类型选择界面

在点击红色加号后，会弹出发布类型选择菜单。

### "写文字"选项
- **位置**：发布类型菜单中的一个选项
- **外观**：文字标签，可能伴有文字图标
- **识别特征**：包含"写文字"或"文字"关键词
- **常见位置**：通常在"拍摄"、"从相册选择"等选项附近

## 普通笔记编辑界面

### "写想法"输入框
- **位置**：选择"写文字"后的主要输入区域
- **外观**：大型文本输入框，通常占据屏幕中上部分
- **识别特征**：
  - 占位符文字可能显示"写想法"、"分享新鲜事"或类似提示
  - 是页面上最大的可输入区域
  - 通常在顶部标题栏下方

### "下一步"按钮
- **位置**：通常在屏幕右上角或底部
- **外观**：文字按钮或带箭头图标的按钮
- **识别特征**：包含"下一步"、"Next"或右箭头图标
- **状态变化**：输入内容后按钮可能从灰色变为可点击的彩色

### 卡片风格选择器
- **位置**：点击第一个"下一步"后的新页面
- **外观**：横向或网格排列的多个卡片样式预览
- **识别特征**：
  - 显示多个不同设计风格的小卡片
  - 可左右滑动或上下滚动浏览
  - 选中的卡片通常有边框高亮或勾选标记

### "添加标题"输入框
- **位置**：选择卡片风格并点击"下一步"后
- **外观**：单行文本输入框
- **识别特征**：
  - 占位符显示"添加标题"或"输入标题"
  - 通常在预览区域上方或下方
  - 比"写想法"输入框小，单行显示

### "发布笔记"按钮
- **位置**：通常在屏幕右上角
- **外观**：醒目的按钮，可能是红色或品牌色
- **识别特征**：
  - 包含"发布"、"发布笔记"或"Publish"文字
  - 通常是页面上最醒目的操作按钮
  - 完成所有必填项后才会激活

## 长文编辑界面

### "写长文"按钮
- **位置**：选择"写文字"后的界面中
- **外观**：文字按钮或带特殊图标的选项
- **识别特征**：
  - 包含"写长文"、"长文"关键词
  - 可能伴有文档或文章图标
  - 与"写想法"选项并列或在附近

### 标题输入框（长文）
- **位置**：长文编辑器顶部
- **外观**：单行文本输入框，字体通常较大
- **识别特征**：
  - 占位符显示"请输入标题"或"标题"
  - 是页面第一个输入框
  - 字号明显大于正文输入区
- **重要提示**：这是一个独立的输入框，仅用于输入标题，不要在此继续输入正文

### 正文输入区（长文）
- **位置**：标题输入框下方，是一个独立的输入框
- **外观**：大型多行文本输入框
- **识别特征**：
  - **标准占位符**："粘贴到这里或输入文字，内容将自动保存"
  - 占据屏幕大部分区域
  - 支持多行输入和换行
  - 与标题输入框有明显分隔
- **操作要点**：
  - 输入标题后，必须点击此输入框才能输入正文
  - **推荐点击位置**：点击占位符文字"粘贴到这里或输入文字，内容将自动保存"偏下方的位置（下方约50-100像素）
  - 避免直接点击占位符文字本身，防止误触到上方的标题输入框
  - 不要在标题输入框中继续输入，否则正文会被错误地添加到标题中
  - 确保光标位于显示占位符"粘贴到这里或输入文字，内容将自动保存"的区域内

### "一键排版"按钮
- **位置**：通常在屏幕右上角
- **外观**：文字按钮或带魔法棒图标的按钮
- **识别特征**：
  - 包含"一键排版"、"排版"或"美化"关键词
  - 可能有魔法棒、画笔等图标
  - 输入内容后才会激活

### 排版样式选择器
- **位置**：点击"一键排版"后显示
- **外观**：多个排版模板的预览图
- **识别特征**：
  - 显示不同字体、配色、布局的样式预览
  - 可滚动浏览多个选项
  - 选中项有高亮或勾选标记

### "添加正文"输入框
- **位置**：选择排版并点击"下一步"后
- **外观**：多行文本输入框
- **识别特征**：
  - 占位符显示"添加正文"、"说点什么"或类似提示
  - 在长文图片预览下方
  - 可输入补充说明文字

## 通用UI识别技巧

### 加载状态识别
- **排版处理中**：可能显示进度条、转圈动画或"处理中"提示
- **等待策略**：检测到加载动画时，应等待其消失后再继续操作

### 错误提示识别
- **弹窗提示**：通常以对话框形式出现在屏幕中央
- **底部提示**：可能在屏幕底部显示toast消息
- **输入框错误**：输入框可能变红色或下方显示错误文字

### 按钮状态识别
- **可点击状态**：按钮颜色鲜明（红色、蓝色等品牌色）
- **不可点击状态**：按钮灰色或半透明
- **加载状态**：按钮上可能显示转圈动画

## 版本兼容性说明

小红书APP的UI可能随版本更新而变化：

1. **文字内容可能变化**：如"写想法"可能改为"记录此刻"等
2. **布局可能调整**：按钮位置可能在不同版本中有所变化
3. **新增功能**：新版本可能添加新的编辑选项

### 应对策略
- 优先使用文字识别（OCR）查找关键词
- 如果精确文字匹配失败，尝试模糊匹配关键词
- 考虑按钮的相对位置（如"右上角"、"底部中央"）
- 遇到识别问题时，向用户描述当前看到的界面并请求指导

## 常见识别问题

### 问题1：找不到红色加号按钮
**可能原因**：
- 不在主界面（可能在其他标签页）
- APP版本过旧或过新，UI变化
- 屏幕分辨率不同导致位置变化

**解决方法**：
- 先导航回主界面（通常点击左下角"首页"）
- 描述当前界面给用户，请求帮助
- 尝试在底部导航栏中央区域查找任何醒目按钮

### 问题2："写文字"选项不显示
**可能原因**：
- 点击加号后菜单还未完全弹出
- 菜单需要滑动才能看到该选项

**解决方法**：
- 等待1-2秒让菜单完全加载
- 尝试上下滑动菜单查找选项
- 查找包含"文字"、"写"、"text"等关键词的选项

### 问题3：输入框无法识别
**可能原因**：
- 输入框未获得焦点
- 需要先点击才能显示输入区域

**解决方法**：
- 点击页面中央区域尝试激活输入框
- 查找最大的空白区域，通常就是输入框
- 检查是否有"点击此处开始输入"类似的提示

### 问题4：长文正文被错误地输入到标题中
**问题描述**：
- 输入标题后，正文内容被接着输入在标题输入框中
- 导致标题过长，正文区域为空

**原因分析**：
- 没有识别出标题和正文是两个独立的输入框
- 输入标题后，光标仍停留在标题输入框中
- 没有正确定位到正文输入框

**解决方法**：
- **关键步骤**：输入标题后，必须定位并点击正文输入框
- **识别方法**：寻找显示"粘贴到这里或输入文字，内容将自动保存"的输入区域
- **操作流程**：
  1. 在标题输入框输入标题
  2. 识别标题下方显示占位符"粘贴到这里或输入文字，内容将自动保存"的正文输入框
  3. **重要**：点击占位符文字偏下方的位置（下方约50-100像素），而非直接点击文字本身
  4. 这样可以确保点击的是正文输入区域，避免误触标题输入框
  5. 确保光标在正文区域后，输入正文内容
- **验证方法**：确认光标位于占位符文字消失、可以输入多行文字的大型输入区域
